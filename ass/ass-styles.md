# 样式参考

* 您可在开始打轴前，将这些样式保存到你的样式库备用，方便快速修改/套用样式
* 以下的样式仅作为参考基准，如感觉画面观感不佳，可自行进行调整（除字体外）
* 样式字幕的颜色，可使用Aegisub自带的取色工具（颜色拾取器）更改

{% hint style="info" %}
样式中用到的字体，请前往[时轴资源](../sources/shi-zhou-zi-yuan.md)自行取用
{% endhint %}

{% hint style="warning" %}
样式可能存在缺失，会逐步完善补全
{% endhint %}
